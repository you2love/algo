概率图模型概述

什么是概率图模型？

概率图模型（Probabilistic Graphical Models）是用图结构表示随机变量之间概率关系的框架。它结合了概率论和图论，能够有效地表示和处理复杂的不确定性推理问题。

主要类型：

  • 贝叶斯网络：有向无环图，表示因果关系
  • 马尔可夫网络：无向图，表示相关性
  • 隐马尔可夫模型：双重随机过程

核心优势：

  • 可视化表示，易于理解
  • 条件独立假设简化计算
  • 支持不确定性推理

朴素贝叶斯分类器 (Naive Bayes)

算法原理

朴素贝叶斯是基于贝叶斯定理的分类器，它假设所有特征相互条件独立（朴素假设），尽管这个假设在现实中往往不成立，但算法仍然非常有效。

贝叶斯定理：

P(C|X) = P(X|C) × P(C) / P(X)

其中：

P(C|X) = 给定特征X时类别C的后验概率

P(X|C) = 给定类别C时特征X的似然

P(C) = 类别C的先验概率

P(X) = 特征X的边缘概率

训练时间

O(n × d)

预测时间

O(c × d)

空间复杂度

O(c × d)

朴素假设的数学表示

条件独立性假设：

P(X₁, X₂, ..., Xₙ | C) = P(X₁|C) × P(X₂|C) × ... × P(Xₙ|C)

分类决策：

Cₘₐₓ = argmax P(C) × ∏ᵢ P(Xᵢ|C)

C

由于P(X)对所有类别相同，可以忽略

拉普拉斯平滑 (Laplace Smoothing)

问题：如果某个特征值在训练集中从未出现，其条件概率为0，会导致整个后验概率为0。

解决方案：给每个计数加上一个平滑参数α

P(Xᵢ = x | C = c) = (count(Xᵢ = x, C = c) + α) / (count(C = c) + α × |Vᵢ|)

其中|Vᵢ|是特征Xᵢ的可能取值数量

常用设置：

  • α = 1：拉普拉斯平滑
  • 0 < α < 1： Lidstone平滑

三种常见的朴素贝叶斯模型

模型 特征类型 应用场景
高斯朴素贝叶斯 连续值 Iris数据集、房价预测
多项式朴素贝叶斯 离散计数 文本分类、词频统计
伯努利朴素贝叶斯 二元特征 垃圾邮件检测、文本分类

class NaiveBayes {
    constructor(alpha = 1) {
        this.alpha = alpha;
        this.classPriors = {};
        this.conditionalProbs = {};
        this.classes = new Set();
        this.featureTypes = {}; // 'categorical', 'continuous'
    }

    fit(X, y, featureTypes = {}) {
        const n = X.length;
        const d = X[0].length;

        // 记录特征类型
        for (let j = 0; j < d; j++) {
            if (!featureTypes[j]) {
                const isContinuous = X.some(x => typeof x[j] === 'number');
                this.featureTypes[j] = isContinuous ? 'continuous' : 'categorical';
            }
        }

        // 统计每个类别的样本数
        const classCounts = {};
        for (const label of y) {
            classCounts[label] = (classCounts[label] || 0) + 1;
            this.classes.add(label);
        }

        // 计算先验概率 P(C)
        for (const c of this.classes) {
            this.classPriors[c] = classCounts[c] / n;
        }

        // 初始化条件概率
        for (const c of this.classes) {
            this.conditionalProbs[c] = {};
        }

        // 计算条件概率 P(Xᵢ|C)
        for (let j = 0; j < d; j++) {
            const type = this.featureTypes[j];
            
            if (type === 'continuous') {
                // 高斯分布参数
                for (const c of this.classes) {
                    this.conditionalProbs[c][j] = { mean: 0, var: 0 };
                }

                const sums = Array(this.classes.size).fill(0);
                const classList = [...this.classes];
                
                for (let i = 0; i < n; i++) {
                    const cIndex = classList.indexOf(y[i]);
                    sums[cIndex] += X[i][j];
                }
                
                const means = sums.map((s, i) => s / classCounts[classList[i]]);
                
                for (let i = 0; i < n; i++) {
                    const cIndex = classList.indexOf(y[i]);
                    const diff = X[i][j] - means[cIndex];
                    this.conditionalProbs[classList[cIndex]][j].mean = means[cIndex];
                    this.conditionalProbs[classList[cIndex]][j].var += diff * diff;
                }

                for (let i = 0; i < classList.length; i++) {
                    this.conditionalProbs[classList[i]][j].var /= (classCounts[classList[i]] - 1 || 1);
                }
            } else {
                // 类别特征：计数
                const valueCounts = {};
                
                for (const c of this.classes) {
                    this.conditionalProbs[c][j] = {};
                    valueCounts[c] = {};
                }

                for (let i = 0; i < n; i++) {
                    const c = y[i];
                    const val = X[i][j];
                    if (!valueCounts[c][val]) {
                        valueCounts[c][val] = 0;
                    }
                    valueCounts[c][val]++;
                }

                // 获取所有可能的值
                const allValues = new Set();
                for (const c of this.classes) {
                    for (const val in valueCounts[c]) {
                        allValues.add(val);
                    }
                }

                for (const c of this.classes) {
                    const total = classCounts[c];
                    for (const val of allValues) {
                        const count = valueCounts[c][val] || 0;
                        // 拉普拉斯平滑
                        this.conditionalProbs[c][j][val] = (count + this.alpha) / (total + this.alpha * allValues.size);
                    }
                }
            }
        }
    }

    _gaussianPDF(x, mean, var) {
        if (var === 0) var = 1e-10;
        const coeff = 1 / Math.sqrt(2 * Math.PI * var);
        const exponent = -Math.pow(x - mean, 2) / (2 * var);
        return coeff * Math.exp(exponent);
    }

    predict(X) {
        return X.map(x => this._predictSingle(x));
    }

    _predictSingle(x) {
        let bestClass = null;
        let bestLogProb = -Infinity;
        const classList = [...this.classes];

        for (const c of classList) {
            let logProb = Math.log(this.classPriors[c]);
            
            for (let j = 0; j < x.length; j++) {
                const type = this.featureTypes[j];
                
                if (type === 'continuous') {
                    const { mean, var: variance } = this.conditionalProbs[c][j];
                    const prob = this._gaussianPDF(x[j], mean, variance);
                    logProb += Math.log(prob + 1e-10);
                } else {
                    const val = String(x[j]);
                    const prob = this.conditionalProbs[c][j][val] || this.alpha / (this.alpha * 10); // 未知值
                    logProb += Math.log(prob);
                }
            }
            
            if (logProb > bestLogProb) {
                bestLogProb = logProb;
                bestClass = c;
            }
        }
        
        return bestClass;
    }

    predictProba(X) {
        return X.map(x => {
            const probs = {};
            const classList = [...this.classes];
            let totalProb = 0;

            for (const c of classList) {
                let logProb = Math.log(this.classPriors[c]);
                
                for (let j = 0; j < x.length; j++) {
                    const type = this.featureTypes[j];
                    
                    if (type === 'continuous') {
                        const { mean, var: variance } = this.conditionalProbs[c][j];
                        const prob = this._gaussianPDF(x[j], mean, variance);
                        logProb += Math.log(prob + 1e-10);
                    } else {
                        const val = String(x[j]);
                        const prob = this.conditionalProbs[c][j][val] || this.alpha / (this.alpha * 10);
                        logProb += Math.log(prob);
                    }
                }
                
                probs[c] = Math.exp(logProb);
                totalProb += probs[c];
            }

            // 归一化
            for (const c of classList) {
                probs[c] /= totalProb;
            }
            
            return probs;
        });
    }
}

import numpy as np
from collections import Counter

class NaiveBayes:
    def __init__(self, alpha=1):
        self.alpha = alpha
        self.class_priors = {}
        self.conditional_probs = {}
        self.classes = set()
        self.feature_types = {}
    
    def fit(self, X, y):
        X = np.array(X)
        y = np.array(y)
        n, d = X.shape
        
        # 检测特征类型
        for j in range(d):
            if np.issubdtype(X[:, j].dtype, np.number):
                self.feature_types[j] = 'continuous'
            else:
                self.feature_types[j] = 'categorical'
        
        # 先验概率
        class_counts = Counter(y)
        for c in class_counts:
            self.class_priors[c] = class_counts[c] / n
            self.classes.add(c)
            self.conditional_probs[c] = {}
        
        # 条件概率
        for j in range(d):
            if self.feature_types[j] == 'continuous':
                # 高斯分布
                for c in self.classes:
                    X_c = X[y == c, j]
                    self.conditional_probs[c][j] = {
                        'mean': np.mean(X_c),
                        'var': np.var(X_c, ddof=1) + 1e-10
                    }
            else:
                # 类别特征
                all_values = set(X[:, j])
                for c in self.classes:
                    X_c = X[y == c, j]
                    count_c = len(X_c)
                    value_counts = Counter(X_c)
                    for val in all_values:
                        count = value_counts.get(val, 0)
                        self.conditional_probs[c][j][val] = (count + self.alpha) / (count_c + self.alpha * len(all_values))
    
    def _gaussian_pdf(self, x, mean, var):
        coeff = 1 / np.sqrt(2 * np.pi * var)
        exponent = -((x - mean) ** 2) / (2 * var)
        return coeff * np.exp(exponent)
    
    def predict(self, X):
        X = np.array(X)
        return [self._predict_single(x) for x in X]
    
    def _predict_single(self, x):
        best_class = None
        best_log_prob = -np.inf
        
        for c in self.classes:
            log_prob = np.log(self.class_priors[c])
            
            for j, val in enumerate(x):
                if self.feature_types[j] == 'continuous':
                    params = self.conditional_probs[c][j]
                    prob = self._gaussian_pdf(val, params['mean'], params['var'])
                    log_prob += np.log(prob + 1e-10)
                else:
                    prob = self.conditional_probs[c][j].get(val, self.alpha / (self.alpha * 10))
                    log_prob += np.log(prob)
            
            if log_prob > best_log_prob:
                best_log_prob = log_prob
                best_class = c
        
        return best_class
    
    def predict_proba(self, X):
        X = np.array(X)
        n = len(X)
        classes = list(self.classes)
        probas = np.zeros((n, len(classes)))
        
        for i, x in enumerate(X):
            log_probs = []
            for c in classes:
                log_prob = np.log(self.class_priors[c])
                for j, val in enumerate(x):
                    if self.feature_types[j] == 'continuous':
                        params = self.conditional_probs[c][j]
                        prob = self._gaussian_pdf(val, params['mean'], params['var'])
                        log_prob += np.log(prob + 1e-10)
                    else:
                        prob = self.conditional_probs[c][j].get(val, self.alpha / (self.alpha * 10))
                        log_prob += np.log(prob)
                log_probs.append(log_prob)
            
            # 归一化
            log_probs = np.array(log_probs)
            log_probs -= np.max(log_probs)
            probs = np.exp(log_probs)
            probs /= probs.sum()
            probs
        
        return probas

package main

import (
    "fmt"
    "math"
)

type NaiveBayes struct {
    alpha            float64
    classPriors      map[string]float64
    conditionalMeans map[string]map[int]float64
    conditionalVars  map[string]map[int]float64
    classes          []string
}

func NewNaiveBayes(alpha float64) *NaiveBayes {
    return &NaiveBayes{
        alpha:            alpha,
        classPriors:      make(map[string]float64),
        conditionalMeans: make(map[string]map[int]float64),
        conditionalVars:  make(map[string]map[int]float64),
    }
}

func (nb *NaiveBayes) Fit(X [][]float64, y []string) {
    n := len(X)
    d := len(X[0])

    // 统计每个类别的样本数
    classCounts := make(map[string]int)
    for _, label := range y {
        classCounts[label]++
    }

    // 计算先验概率
    nb.classes = make([]string, 0, len(classCounts))
    for class, count := range classCounts {
        nb.classPriors[class] = float64(count) / float64(n)
        nb.classes = append(nb.classes, class)
        nb.conditionalMeans[class] = make(map[int]float64)
        nb.conditionalVars[class] = make(map[int]float64)
    }

    // 计算每个类别每个特征的均值和方差
    for _, class := range nb.classes {
        for j := 0; j < d; j++ {
            var sum float64
            count := 0
            for i := 0; i < n; i++ {
                if y[i] == class {
                    sum += X[i][j]
                    count++
                }
            }
            mean := sum / float64(count)
            nb.conditionalMeans[class][j] = mean

            // 计算方差
            var varSum float64
            for i := 0; i < n; i++ {
                if y[i] == class {
                    diff := X[i][j] - mean
                    varSum += diff * diff
                }
            }
            nb.conditionalVars[class][j] = varSum / float64(count-1) + 1e-10
        }
    }
}

func (nb *NaiveBayes) gaussianPDF(x, mean, variance float64) float64 {
    coeff := 1.0 / math.Sqrt(2*math.Pi*variance)
    exponent := -math.Pow(x-mean, 2) / (2 * variance)
    return coeff * math.Exp(exponent)
}

func (nb *NaiveBayes) Predict(X [][]float64) []string {
    predictions := make([]string, len(X))
    for i, x := range X {
        predictions[i] = nb.predictSingle(x)
    }
    return predictions
}

func (nb *NaiveBayes) predictSingle(x []float64) string {
    var bestClass string
    bestLogProb := math.Inf(-1)

    for _, class := range nb.classes {
        logProb := math.Log(nb.classPriors[class])

        for j, val := range x {
            mean := nb.conditionalMeans[class][j]
            variance := nb.conditionalVars[class][j]
            prob := nb.gaussianPDF(val, mean, variance)
            logProb += math.Log(prob + 1e-10)
        }

        if logProb > bestLogProb {
            bestLogProb = logProb
            bestClass = class
        }
    }

    return bestClass
}

朴素贝叶斯的优缺点

优点：

  • 简单高效，训练和预测速度快
  • 对小样本数据效果好
  • 抗过拟合能力强
  • 对缺失值不敏感

缺点：

  • 条件独立性假设过强
  • 特征间有相关性时效果下降
  • 连续特征假设正态分布可能不准确

朴素贝叶斯应用场景

  • 文本分类：垃圾邮件检测、情感分析、新闻分类
  • 多分类问题：类别数量较多时表现良好
  • 实时预测：计算量小，适合在线场景
  • 特征工程：作为基线模型

隐马尔可夫模型 (HMM)

算法原理

隐马尔可夫模型（Hidden Markov Model, HMM）是一种统计模型，用于描述一个含有隐含状态（未观测）的马尔可夫过程。它包含两个随机过程：

  • 隐状态链：不可观测的马尔可夫链
  • 观测序列：由隐状态生成的观测值

HMM的五元组定义：

λ = (N, M, A, B, π)

其中：

N = 隐状态数量

M = 观测符号数量

A = 状态转移概率矩阵 (N×N)

B = 观测概率矩阵 (N×M)

π = 初始状态概率分布 (N)

评估问题

O(T × N²)

解码问题

O(T × N²)

学习问题

O(T × N² × I)

HMM的三个基本问题

问题1：评估（概率计算）

给定模型λ和观测序列O，计算P(O|λ)。

解法：前向算法或后向算法

问题2：解码（状态预测）

给定模型λ和观测序列O，找到最可能的隐状态序列Q。

解法：Viterbi算法

问题3：学习（参数估计）

给定观测序列O，估计模型参数λ使得P(O|λ)最大。

解法：Baum-Welch算法（EM算法特例）

前向算法（解决评估问题）

前向变量：

αₜ(i) = P(o₁, o₂, ..., oₜ, qₜ = Sᵢ | λ)

表示到时刻t为止的观测序列，且在状态Sᵢ的概率

递归步骤：

  1. 初始化：α₁(i) = πᵢ × bᵢ(o₁)
  2. 递归：αₜ₊₁(j) = [Σᵢ αₜ(i) × aᵢⱼ] × bⱼ(oₜ₊₁)
  3. 终止：P(O|λ) = Σᵢ αₜ(i)

时间复杂度：O(T × N²)

相比直接计算O(T × Nᵀ)的复杂度大幅降低

Viterbi算法（解决解码问题）

Viterbi变量：

δₜ(i) = max P(q₁, q₂, ..., qₜ = Sᵢ, o₁, o₂, ..., oₜ | λ)

表示到时刻t为止，最可能产生观测序列的状态序列，且当前状态为Sᵢ的概率

递归步骤：

  1. 初始化：δ₁(i) = πᵢ × bᵢ(o₁), ψ₁(i) = 0
  2. 递归：δₜ(j) = maxᵢ [δₜ₋₁(i) × aᵢⱼ] × bⱼ(oₜ)
  3. 回溯：从T时刻回溯最优路径

回溯公式：

qₜ* = argmaxᵢ [δₜ(i)]

qₜ₋₁* = ψₜ(qₜ*)

Baum-Welch算法（解决学习问题）

Baum-Welch是EM算法在HMM中的特例，通过迭代优化期望最大化来估计参数。

辅助函数Q(λ, λ̄)：

Q(λ̄, λ) = Σ_Q P(Q|O, λ) × log P(O, Q|λ̄)

E步：计算期望

使用前向后向算法计算γ和ξ：

γₜ(i) = P(qₜ = Sᵢ | O, λ) = αₜ(i) × βₜ(i) / P(O|λ)

ξₜ(i, j) = P(qₜ = Sᵢ, qₜ₊₁ = Sⱼ | O, λ)

= αₜ(i) × aᵢⱼ × bⱼ(oₜ₊₁) × βₜ₊₁(j) / P(O|λ)

M步：最大化

πᵢ = γ₁(i)

aᵢⱼ = Σₜ ξₜ(i, j) / Σₜ γₜ(i)

bⱼ(k) = Σₜ γₜ(j) × δ(oₜ, vₖ) / Σₜ γₜ(j)

算法实现


class HMM {
    constructor(nStates, nObservations) {
        this.nStates = nStates;
        this.nObservations = nObservations;
        this.A = this._randomMatrix(nStates, nStates);
        this.B = this._randomMatrix(nStates, nObservations);
        this.pi = new Array(nStates).fill(1 / nStates);
    }

    _randomMatrix(rows, cols) {
        const mat = [];
        for (let i = 0; i < rows; i++) {
            const row = [];
            let sum = 0;
            for (let j = 0; j < cols; j++) {
                const val = Math.random();
                row.push(val);
                sum += val;
            }
            mat.push(row.map(v => v / sum));
        }
        return mat;
    }

    // 前向算法
    forward(observations) {
        const T = observations.length;
        const alpha = Array(T).fill(null).map(() => new Array(this.nStates).fill(0));

        // 初始化
        for (let i = 0; i < this.nStates; i++) {
            alpha[0][i] = this.pi[i] * this.B[i][observations[0]];
        }

        // 递归
        for (let t = 1; t < T; t++) {
            for (let j = 0; j < this.nStates; j++) {
                let sum = 0;
                for (let i = 0; i < this.nStates; i++) {
                    sum += alpha[t-1][i] * this.A[i][j];
                }
                alpha[t][j] = sum * this.B[j][observations[t]];
            }
        }

        // 终止：计算观测序列的概率
        const prob = alpha[T-1].reduce((sum, val) => sum + val, 0);
        return { alpha, prob };
    }

    // 后向算法
    backward(observations) {
        const T = observations.length;
        const beta = Array(T).fill(null).map(() => new Array(this.nStates).fill(0));

        // 初始化
        for (let i = 0; i < this.nStates; i++) {
            beta[T-1][i] = 1;
        }

        // 递归
        for (let t = T-2; t >= 0; t--) {
            for (let i = 0; i < this.nStates; i++) {
                let sum = 0;
                for (let j = 0; j < this.nStates; j++) {
                    sum += this.A[i][j] * this.B[j][observations[t+1]] * beta[t+1][j];
                }
                beta[t][i] = sum;
            }
        }

        return beta;
    }

    // Viterbi算法：解码
    viterbi(observations) {
        const T = observations.length;
        const delta = Array(T).fill(null).map(() => new Array(this.nStates).fill(0));
        const psi = Array(T).fill(null).map(() => new Array(this.nStates).fill(0));

        // 初始化
        for (let i = 0; i < this.nStates; i++) {
            delta[0][i] = this.pi[i] * this.B[i][observations[0]];
            psi[0][i] = 0;
        }

        // 递归
        for (let t = 1; t < T; t++) {
            for (let j = 0; j < this.nStates; j++) {
                let maxProb = -1;
                let maxState = -1;
                
                for (let i = 0; i < this.nStates; i++) {
                    const prob = delta[t-1][i] * this.A[i][j];
                    if (prob > maxProb) {
                        maxProb = prob;
                        maxState = i;
                    }
                }
                
                delta[t][j] = maxProb * this.B[j][observations[t]];
                psi[t][j] = maxState;
            }
        }

        // 回溯
        const path = new Array(T);
        let maxProb = -1;
        let maxState = -1;
        
        for (let i = 0; i < this.nStates; i++) {
            if (delta[T-1][i] > maxProb) {
                maxProb = delta[T-1][i];
                maxState = i;
            }
        }
        
        path[T-1] = maxState;
        
        for (let t = T-2; t >= 0; t--) {
            path[t] = psi[t+1][path[t+1]];
        }

        return { path, prob: maxProb };
    }

    // Baum-Welch算法：学习
    baumWelch(observations, maxIter = 100, tol = 1e-6) {
        const T = observations.length;
        
        for (let iter = 0; iter < maxIter; iter++) {
            // E步
            const { alpha, prob } = this.forward(observations);
            const beta = this.backward(observations);
            
            // 计算gamma和xi
            const gamma = Array(T).fill(null).map(() => new Array(this.nStates).fill(0));
            const xi = Array(T-1).fill(null).map(() => 
                this.nStates > 0 ? new Array(this.nStates * this.nStates).fill(0) : []
            );

            for (let t = 0; t < T; t++) {
                let sum = 0;
                for (let i = 0; i < this.nStates; i++) {
                    gamma[t][i] = alpha[t][i] * (t < T-1 ? beta[t][i] : 1);
                    sum += gamma[t][i];
                }
                for (let i = 0; i < this.nStates; i++) {
                    gamma[t][i] /= (sum || 1);
                }
            }

            for (let t = 0; t < T-1; t++) {
                let sum = 0;
                for (let i = 0; i < this.nStates; i++) {
                    for (let j = 0; j < this.nStates; j++) {
                        const val = alpha[t][i] * this.A[i][j] * 
                                   this.B[j][observations[t+1]] * beta[t+1][j];
                        xi[t][i * this.nStates + j] = val;
                        sum += val;
                    }
                }
                for (let i = 0; i < this.nStates; i++) {
                    for (let j = 0; j < this.nStates; j++) {
                        xi[t][i * this.nStates + j] /= (sum || 1);
                    }
                }
            }

            // M步：更新参数
            const oldPi = [...this.pi];
            const oldA = this.A.map(row => [...row]);
            const oldB = this.B.map(row => [...row]);

            // 更新pi
            for (let i = 0; i < this.nStates; i++) {
                this.pi[i] = gamma[0][i];
            }

            // 更新A
            for (let i = 0; i < this.nStates; i++) {
                let gammaSum = 0;
                for (let t = 0; t < T-1; t++) {
                    gammaSum += gamma[t][i];
                }
                
                for (let j = 0; j < this.nStates; j++) {
                    let xiSum = 0;
                    for (let t = 0; t < T-1; t++) {
                        xiSum += xi[t][i * this.nStates + j];
                    }
                    this.A[i][j] = xiSum / (gammaSum || 1);
                }
            }

            // 更新B
            for (let j = 0; j < this.nStates; j++) {
                let gammaSum = 0;
                for (let t = 0; t < T; t++) {
                    gammaSum += gamma[t][j];
                }
                
                for (let k = 0; k < this.nObservations; k++) {
                    let obsGammaSum = 0;
                    for (let t = 0; t < T; t++) {
                        if (observations[t] === k) {
                            obsGammaSum += gamma[t][j];
                        }
                    }
                    this.B[j][k] = obsGammaSum / (gammaSum || 1);
                }
            }

            // 检查收敛
            const piDiff = Math.abs(this.pi.reduce((s, v, i) => s + Math.abs(v - oldPi[i]), 0));
            const aDiff = this.A.reduce((s, row, i) => 
                s + row.reduce((ss, v, j) => ss + Math.abs(v - oldA[i][j]), 0), 0);
            
            if (piDiff + aDiff < tol) {
                break;
            }
        }
    }
}

import numpy as np

class HMM:
    def __init__(self, n_states, n_observations):
        self.n_states = n_states
        self.n_observations = n_observations
        self.A = np.random.rand(n_states, n_states)
        self.A /= self.A.sum(axis=1, keepdims=True)
        self.B = np.random.rand(n_states, n_observations)
        self.B /= self.B.sum(axis=1, keepdims=True)
        self.pi = np.ones(n_states) / n_states
    
    def forward(self, observations):
        T = len(observations)
        alpha = np.zeros((T, self.n_states))
        
        # 初始化
        alpha[0] = self.pi * self.B[:, observations[0]]
        
        # 递归
        for t in range(1, T):
            for j in range(self.n_states):
                alpha[t, j] = np.sum(alpha[t-1] * self.A[:, j]) * self.B[j, observations[t]]
        
        prob = np.sum(alpha[T-1])
        return alpha, prob
    
    def backward(self, observations):
        T = len(observations)
        beta = np.zeros((T, self.n_states))
        
        # 初始化
        beta[T-1] = 1
        
        # 递归
        for t in range(T-2, -1, -1):
            for i in range(self.n_states):
                beta[t, i] = np.sum(self.A[i, :] * self.B[:, observations[t+1]] * beta[t+1])
        
        return beta
    
    def viterbi(self, observations):
        T = len(observations)
        delta = np.zeros((T, self.n_states))
        psi = np.zeros((T, self.n_states), dtype=int)
        
        # 初始化
        delta[0] = self.pi * self.B[:, observations[0]]
        psi[0] = 0
        
        # 递归
        for t in range(1, T):
            for j in range(self.n_states):
                delta[t, j] = np.max(delta[t-1] * self.A[:, j]) * self.B[j, observations[t]]
                psi[t, j] = np.argmax(delta[t-1] * self.A[:, j])
        
        # 回溯
        path = np.zeros(T, dtype=int)
        path[T-1] = np.argmax(delta[T-1])
        for t in range(T-2, -1, -1):
            path[t] = psi[t+1, path[t+1]]
        
        return path, np.max(delta[T-1])
    
    def baum_welch(self, observations, max_iter=100, tol=1e-6):
        T = len(observations)
        
        for _ in range(max_iter):
            alpha, prob = self.forward(observations)
            beta = self.backward(observations)
            
            # E步：计算gamma和xi
            gamma = alpha * beta
            gamma /= (gamma.sum(axis=1, keepdims=True) + 1e-10)
            
            xi = np.zeros((T-1, self.n_states, self.n_states))
            for t in range(T-1):
                for i in range(self.n_states):
                    for j in range(self.n_states):
                        xi[t, i, j] = alpha[t, i] * self.A[i, j] * \
                                     self.B[j, observations[t+1]] * beta[t+1, j]
                xi[t] /= (xi[t].sum() + 1e-10)
            
            # M步：更新参数
            old_pi = self.pi.copy()
            old_A = self.A.copy()
            
            self.pi = gamma[0]
            
            for i in range(self.n_states):
                gamma_sum = gamma[:T-1, i].sum()
                self.A[i] = xi[:T-1, i].sum(axis=0) / (gamma_sum + 1e-10)
            
            for j in range(self.n_states):
                gamma_sum = gamma[:, j].sum()
                for k in range(self.n_observations):
                    obs_gamma = np.where(observations == k)[0]
                    self.B[j, k] = gamma[obs_gamma, j].sum() / (gamma_sum + 1e-10)
            
            # 检查收敛
            pi_diff = np.abs(self.pi - old_pi).sum()
            a_diff = np.abs(self.A - old_A).sum()
            
            if pi_diff + a_diff < tol:
                break

HMM的优缺点

优点：

  • 理论基础扎实，数学表达优雅
  • 高效的概率计算和状态预测算法
  • 参数学习有成熟的EM算法
  • 可解释性强，隐状态有实际意义

缺点：

  • 马尔可夫假设过强（当前状态只与前一状态相关）
  • 观测独立假设（观测值只与当前状态相关）
  • 状态空间离散且有限
  • 对长序列可能出现数值下溢

HMM的应用场景

  • 语音识别：将声学特征序列转换为文本
  • 词性标注：给定句子，确定每个词的词性
  • 生物信息学：基因序列分析、蛋白质结构预测
  • 金融分析：市场状态预测、期权定价
  • 手势识别：传感器数据序列的意图识别
  • 网络安全：异常行为检测

算法对比与选择

朴素贝叶斯 vs HMM

特性 朴素贝叶斯 HMM
模型类型 生成式（判别式变体存在） 生成式
时间依赖 无 有（马尔可夫链）
隐变量 无 有（隐状态）
输出 类别标签 类别标签 + 隐状态序列
序列建模 不支持 支持

选择建议

  • 选择朴素贝叶斯当：
    • 数据特征条件独立或近似独立
    • 需要快速训练和预测
    • 样本量较小
    • 多分类问题
  • 选择HMM当：
    • 数据有明显的序列依赖关系
    • 需要建模隐含状态
    • 观测序列长度适中（不太长）
    • 需要解码最优状态序列
