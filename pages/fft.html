快速傅里叶变换 (Fast Fourier Transform)

快速傅里叶变换是数字信号处理领域最核心的算法之一，用于高效计算离散傅里叶变换。

1. 概述

什么是傅里叶变换？

傅里叶变换是一种将信号从时域转换到频域的数学变换。任何周期信号都可以表示为不同频率正弦波的叠加。

f(t) = A₀ + Σ[Aₙcos(nω₀t) + Bₙsin(nω₀t)]

为什么需要FFT？

直接计算DFT的时间复杂度为O(n²)，而FFT将其降低到O(nlogn)，这使得实时信号处理成为可能。

方法 时间复杂度 1000点计算量 100万点计算量
直接DFT O(n²) 100万次 10¹²次
FFT O(nlogn) 约1万次 约2000万次

历史背景

FFT最早由Cooley和Tukey于1965年发表，但实际上Gauss在1805年就提出了类似的思想。

2. 离散傅里叶变换 (DFT)

DFT定义

对于长度为N的离散序列x[n]，其DFT定义为：

X[k] = Σ_{n=0}^{N-1} x[n] · e^(-j2πkn/N), k = 0, 1, ..., N-1

使用旋转因子表示：

X[k] = Σ_{n=0}^{N-1} x[n] · W_N^{kn}

其中旋转因子：

W_N = e^(-j2π/N) = cos(2π/N) - j·sin(2π/N)

旋转因子的性质

性质 公式 说明
周期性 W_N^{k+N} = W_N^k 周期为N
对称性 W_N^{k+N/2} = -W_N^k 共轭对称
可约性 W_N^{kn} = W_{N/d}^{k'n} 因子分解
逆变换 (W_N^{kn})* = W_N^{-kn} 共轭关系

DFT示例

对简单序列x[n] = [1, 1, 1, 1]（直流信号）进行DFT：

X[0] = 1+1+1+1 = 4（直流分量）
X[1] = 1-j+(-1)+j = 0
X[2] = 1+(-1)+1+(-1) = 0
X[3] = 1+j+(-1)+(-j) = 0

Python实现DFT

Python
import numpy as np
import cmath

def dft(x):
    """直接计算离散傅里叶变换"""
    n = len(x)
    X = []
    
    for k in range(n):
        sum_val = 0j
        for n_val in range(n):
            angle = -2 * np.pi * k * n_val / n
            sum_val += x[n_val] * complex(np.cos(angle), np.sin(angle))
        X.append(sum_val)
    
    return X

def idft(X):
    """计算逆离散傅里叶变换"""
    n = len(X)
    x = []
    
    for k in range(n):
        sum_val = 0j
        for n_val in range(n):
            angle = 2 * np.pi * k * n_val / n
            sum_val += X[n_val] * complex(np.cos(angle), np.sin(angle))
        x.append(sum_val / n)
    
    return x

X = dft(x)

# [4, 0, 0, 0] - 只有直流分量

# 验证逆变换

print("逆DFT结果:", x_reconstructed)

3. FFT原理

核心思想：分治策略

FFT利用旋转因子的对称性和周期性，将长序列的DFT分解为短序列的DFT计算。

基2时间抽取(DIT)FFT

假设序列长度N是2的幂（N=2^M），将输入序列按奇偶分组：

X[k] = Σ_{n=0}^{N-1} x[n] · W_N^{kn}
= Σ_{r=0}^{N/2-1} x[2r] · W_N^{k(2r)} + Σ_{r=0}^{N/2-1} x[2r+1] · W_N^{k(2r+1)}
= Σ_{r=0}^{N/2-1} x[2r] · W_{N/2}^{kr} + W_N^k · Σ_{r=0}^{N/2-1} x[2r+1] · W_{N/2}^{kr}

分解过程

8点DFT分解为4点DFT的计算流程：

N=8 → N/2=4 → N/4=2 → N/8=1（1点DFT等于自身）

完整推导

设偶序列为E[r] = x[2r]，奇序列为O[r] = x[2r+1]：

X[k] = E[k mod (N/2)] + W_N^k · O[k mod (N/2)]

其中E[k]和O[k]分别是偶序列和奇序列的(N/2)点DFT。

基2频率抽取(DIF)FFT

与时间抽取类似，但在频域进行分组：

X[k] = Σ_{n=0}^{N/2-1} x[n] · W_N^{kn} + Σ_{n=0}^{N/2-1} x[n+N/2] · W_N^{k(n+N/2)}
= Σ_{n=0}^{N/2-1} [x[n] + x[n+N/2] · W_N^{kN/2}] · W_N^{kn}

4. 蝶形运算

蝶形单元

蝶形运算是FFT的基本计算单元，一次蝶形运算完成两个复数的加权和计算。

A' = A + W · B
B' = A - W · B

Python实现基2 DIT FFT

Python
import numpy as np
import cmath

def fft(x):
    """快速傅里叶变换 - 基2时间抽取算法"""
    n = len(x)
    
    # 递归基本情况：长度为1的FFT等于自身
    if n == 1:
        return x
    
    # 检查n是否为2的幂
    if n % 2 != 0:
        raise ValueError("序列长度必须是2的幂")
    
    # 偶序列和奇序列
    even = fft(x[::2])
    odd = fft(x[1::2])
    
    # 合并结果
    X = [0j] * n
    for k in range(n // 2):
        # 旋转因子
        angle = -2 * np.pi * k / n
        w = complex(np.cos(angle), np.sin(angle))
        
        # 蝶形运算
        X[k] = even[k] + w * odd[k]
        X[k + n // 2] = even[k] - w * odd[k]
    
    return X

def ifft(X):
    """快速逆傅里叶变换"""
    n = len(X)
    
    # 取共轭，FFT，取共轭，除以n
    x_conj = [complex(c.real, -c.imag) for c in X]
    x_fft = fft(x_conj)
    x = [complex(c.real / n, -c.imag / n) for c in x_fft]
    
    return x

def fft_iterative(x):
    """迭代版FFT - 使用位逆序重排"""
    n = len(x)
    if n % 2 != 0:
        raise ValueError("序列长度必须是2的幂")
    
    # 复制输入
    X = x.copy()
    
    # 位逆序重排
    j = 0
    for i in range(1, n):
        bit = n >> 1
        while j & bit:
            j ^= bit
            bit >>= 1
        j ^= bit
        if i < j:
            X[i], X[j] = X[j], X[i]
    
    # 逐级蝶形运算
    length = 2
    while length <= n:
        half = length // 2
        for i in range(0, n, length):
            for j in range(half):
                # 旋转因子
                angle = -2 * np.pi * j / length
                w = complex(np.cos(angle), np.sin(angle))
                
                # 蝶形运算
                u = X[i + j]
                v = w * X[i + j + half]
                X[i + j] = u + v
                X[i + j + half] = u - v
        length *= 2
    
    return X

JavaScript实现FFT

JavaScript
class FFT {
    constructor() {
        this.sinTable = [];
        this.cosTable = [];
    }

    // 快速傅里叶变换 - 递归版
    fft(x) {
        const n = x.length;
        
        if (n === 1) {
            return x;
        }
        
        if (n % 2 !== 0) {
            throw new Error("序列长度必须是2的幂");
        }
        
        const even = this.fft(x.filter((_, i) => i % 2 === 0));
        const odd = this.fft(x.filter((_, i) => i % 2 === 1));
        
        const X = new Array(n).fill(0);
        
        for (let k = 0; k < n / 2; k++) {
            const angle = -2 * Math.PI * k / n;
            const w = {
                real: Math.cos(angle),
                imag: Math.sin(angle)
            };
            
            // 复数乘法: w * odd[k]
            const oddTerm = {
                real: w.real * odd[k].real - w.imag * odd[k].imag,
                imag: w.real * odd[k].imag + w.imag * odd[k].real
            };
            
            X[k] = {
                real: even[k].real + oddTerm.real,
                imag: even[k].imag + oddTerm.imag
            };
            
            X[k + n / 2] = {
                real: even[k].real - oddTerm.real,
                imag: even[k].imag - oddTerm.imag
            };
        }
        
        return X;
    }

    // 快速逆傅里叶变换
    ifft(X) {
        const n = X.length;
        
        // 取共轭
        const X_conj = X.map(c => ({ real: c.real, imag: -c.imag }));
        
        // FFT
        const x_fft = this.fft(X_conj);
        
        // 取共轭并除以n
        return x_fft.map(c => ({
            real: c.real / n,
            imag: -c.imag / n
        }));
    }

    // 迭代版FFT - 更高效
    fftIterative(x) {
        const n = x.length;
        
        if ((n & (n - 1)) !== 0) {
            throw new Error("序列长度必须是2的幂");
        }
        
        // 复制输入
        const X = x.map(c => ({ real: c.real !== undefined ? c.real : c, imag: 0 }));
        
        // 位逆序重排
        let j = 0;
        for (let i = 1; i < n; i++) {
            let bit = n >> 1;
            while (j & bit) {
                j ^= bit;
                bit >>= 1;
            }
            j ^= bit;
            if (i < j) {
                [X[i], X[j]] = [X[j], X[i]];
            }
        }
        
        // 逐级蝶形运算
        for (let length = 2; length <= n; length *= 2) {
            const half = length / 2;
            for (let i = 0; i < n; i += length) {
                for (let j = 0; j < half; j++) {
                    const angle = -2 * Math.PI * j / length;
                    const w = {
                        real: Math.cos(angle),
                        imag: Math.sin(angle)
                    };
                    
                    const u = X[i + j];
                    const v = {
                        real: w.real * X[i + j + half].real - w.imag * X[i + j + half].imag,
                        imag: w.real * X[i + j + half].imag + w.imag * X[i + j + half].real
                    };
                    
                    X[i + j] = {
                        real: u.real + v.real,
                        imag: u.imag + v.imag
                    };
                    
                    X[i + j + half] = {
                        real: u.real - v.real,
                        imag: u.imag - v.imag
                    };
                }
            }
        }
        
        return X;
    }

    // 计算幅值谱
    magnitudeSpectrum(X) {
        return X.map(c => Math.sqrt(c.real * c.real + c.imag * c.imag));
    }

    // 计算功率谱
    powerSpectrum(X) {
        return X.map(c => {
            const mag = Math.sqrt(c.real * c.real + c.imag * c.imag);
            return mag * mag;
        });
    }
}

// 测试
function testFFT() {
    const fft = new FFT();
    
    // 生成测试信号: sin(2π·50t) + 0.5·sin(2π·120t)
    const fs = 1000; // 采样频率
    const signal = [];
    for (let n = 0; n < 256; n++) {
        const t = n / fs;
        signal.push({
            real: Math.sin(2 * Math.PI * 50 * t) + 0.5 * Math.sin(2 * Math.PI * 120 * t),
            imag: 0
        });
    }
    
    // 计算FFT
    const X = fft.fftIterative(signal);
    const magnitudes = fft.magnitudeSpectrum(X);
    
    // 找到峰值频率
    let maxMag = 0;
    let peakFreq = 0;
    for (let i = 1; i < magnitudes.length / 2; i++) {
        if (magnitudes[i] > maxMag) {
            maxMag = magnitudes[i];
            peakFreq = i * fs / magnitudes.length;
        }
    }
    
    console.log("信号长度:", signal.length);
    console.log("频率分辨率:", fs / signal.length, "Hz");
    console.log("检测到的峰值频率:", peakFreq.toFixed(1), "Hz");
    
    // 验证逆变换
    const x_reconstructed = fft.ifft(X);
    console.log("逆变换重建误差:", 
        Math.sqrt(signal.reduce((sum, c, i) => {
            const diff = c.real - x_reconstructed[i].real;
            return sum + diff * diff;
        }, 0) / signal.length));
}

// 运行测试
testFFT();

Go语言实现

Go
package main

import (
    "fmt"
    "math"
)

type Complex struct {
    real float64
    imag float64
}

func (c Complex) Add(other Complex) Complex {
    return Complex{c.real + other.real, c.imag + other.imag}
}

func (c Complex) Sub(other Complex) Complex {
    return Complex{c.real - other.real, c.imag - other.imag}
}

func (c Complex) Mul(other Complex) Complex {
    return Complex{
        c.real*other.real - c.imag*other.imag,
        c.real*other.imag + c.imag*other.real,
    }
}

func FFT(x []Complex) []Complex {
    n := len(x)
    
    if n == 1 {
        return x
    }
    
    if n%2 != 0 {
        panic("序列长度必须是2的幂")
    }
    
    // 偶序列和奇序列
    var even []Complex
    var odd []Complex
    for i := 0; i < n; i++ {
        if i%2 == 0 {
            even = append(even, x[i])
        } else {
            odd = append(odd, x[i])
        }
    }
    
    evenFFT := FFT(even)
    oddFFT := FFT(odd)
    
    X := make([]Complex, n)
    
    for k := 0; k < n/2; k++ {
        angle := -2 * math.Pi * float64(k) / float64(n)
        w := Complex{math.Cos(angle), math.Sin(angle)}
        
        X[k] = evenFFT[k].Add(w.Mul(oddFFT[k]))
        X[k+n/2] = evenFFT[k].Sub(w.Mul(oddFFT[k]))
    }
    
    return X
}

func IFFT(X []Complex) []Complex {
    n := len(X)
    
    // 取共轭
    var X_conj []Complex
    for _, c := range X {
        X_conj = append(X_conj, Complex{c.real, -c.imag})
    }
    
    // FFT
    x_fft := FFT(X_conj)
    
    // 取共轭并除以n
    var x []Complex
    for _, c := range x_fft {
        x = append(x, Complex{c.real / float64(n), -c.imag / float64(n)})
    }
    
    return x
}

func main() {
    // 生成测试信号: sin(2π·50t) + 0.5·sin(2π·120t)
    fs := 1000.0 // 采样频率
    n := 256
    
    var signal []Complex
    for i := 0; i < n; i++ {
        t := float64(i) / fs
        val := math.Sin(2*math.Pi*50*t) + 0.5*math.Sin(2*math.Pi*120*t)
        signal = append(signal, Complex{val, 0})
    }
    
    // 计算FFT
    X := FFT(signal)
    
    // 找峰值
    var maxMag float64
    var peakIdx int
    for i := 1; i < n/2; i++ {
        mag := math.Sqrt(X[i].real*X[i].real + X[i].imag*X[i].imag)
        if mag > maxMag {
            maxMag = mag
            peakIdx = i
        }
    }
    
    freqResolution := fs / float64(n)
    fmt.Printf("频率分辨率: %.2f Hz\n", freqResolution)
    fmt.Printf("峰值频率: %.1f Hz\n", float64(peakIdx)*freqResolution)
    
    // 验证逆变换
    x_reconstructed := IFFT(X)
    var mse float64
    for i := 0; i < n; i++ {
        diff := signal[i].real - x_reconstructed[i].real
        mse += diff * diff
    }
    mse /= float64(n)
    fmt.Printf("重建MSE: %.10f\n", mse)
}

import numpy as np
from scipy import signal

# OFDM示例
def ofdm_modulate(data, n_subcarriers=64, n_cyclic_prefix=16):
    """OFDM调制"""
    # IFFT
    ofdm_symbols = np.fft.ifft(data) * np.sqrt(n_subcarriers)
    
    # 添加循环前缀
    cp = ofdm_symbols[-n_cyclic_prefix:]
    ofdm_tx = np.concatenate([cp, ofdm_symbols])
    
    return ofdm_tx

def ofdm_demodulate(rx_signal, n_subcarriers=64, n_cyclic_prefix=16):
    """OFDM解调"""
    # 去除循环前缀
    ofdm_symbols = rx_signal[n_cyclic_prefix:n_cyclic_prefix + n_subcarriers]
    
    # FFT
    data_rx = np.fft.fft(ofdm_symbols) / np.sqrt(n_subcarriers)
    
    return data_rx

# QPSK调制 + OFDM
def qpsk_modulate(bits):
    """QPSK调制"""
    # 比特到符号映射
    symbols = np.zeros(len(bits) // 2, dtype=complex)
    for i in range(0, len(bits), 2):
        b1, b2 = bits[i], bits[i+1]
        if b1 == 0 and b2 == 0:
            symbols[i//2] = (1 + 1j) / np.sqrt(2)
        elif b1 == 0 and b2 == 1:
            symbols[i//2] = (-1 + 1j) / np.sqrt(2)
        elif b1 == 1 and b2 == 0:
            symbols[i//2] = (1 - 1j) / np.sqrt(2)
        else:
            symbols[i//2] = (-1 - 1j) / np.sqrt(2)
    return symbols

def fft_convolution(x, h):
    """使用FFT计算线性卷积"""
    n = len(x) + len(h) - 1
    
    # 补零到2的幂
    N = 1
    while N < n:
        N *= 2

    H = np.fft.fft(h, N)
    
    Y = X * H
    
    y = np.fft.ifft(Y).real[:n]
    
    return y

7. 逆FFT

逆FFT公式

x[n] = (1/N) · Σ_{k=0}^{N-1} X[k] · e^(j2πkn/N)

与FFT相比，逆FFT有三点不同：

  • 旋转因子取正号：W_N^{-kn}
  • 最后结果除以N
  • 可通过对FFT结果取共轭实现

IFFT的实现方法

IDFT[x[n]] = (1/N) · conj(DFT[conj(x[n])])

Python实现IFFT

Python
def ifft(X):
    """方法1: 使用共轭性质"""
    n = len(X)
    
    # 取共轭
    X_conj = np.conj(X)
    
    # FFT
    x_fft = np.fft.fft(X_conj)
    
    # 取共构并除以n
    x = np.conj(x_fft) / n
    
    return x

def ifft_direct(X):
    """方法2: 直接计算"""
    n = len(X)
    x = np.zeros(n, dtype=complex)
    
    for k in range(n):
        sum_val = 0j
        for m in range(n):
            angle = 2 * np.pi * k * m / n
            sum_val += X[m] * (np.cos(angle) + 1j * np.sin(angle))
        x[k] = sum_val / n
    
    return x

# 验证两种方法结果一致
X = np.random.randn(8) + 1j * np.random.randn(8)
x1 = ifft(X)
x2 = ifft_direct(X)
print("两种IFFT方法差异:", np.max(np.abs(x1 - x2)))

8. 算法对比

FFT变体对比

算法 时间复杂度 适用场景 特点
基2 DIT FFT O(nlogn) 序列长度N=2^M 输入需位逆序
基2 DIF FFT O(nlogn) 序列长度N=2^M 输出需位逆序
基4 FFT O(nlogn) 序列长度N=4^M 减少乘法次数
分裂基 FFT O(nlogn) 通用 计算量最少
Bluestein算法 O(nlogn) 任意长度N 适合素数长度

复杂度分析

N DFT O(n²) FFT O(nlogn) 加速比
8 64 24 2.7x
64 4096 384 10.7x
256 65536 2048 32x
1024 104万 10240 102x
65536 43亿 104万 4096x

实际考虑因素

  • 缓存效率：迭代版比递归版更高效
  • 浮点精度：大N时注意数值误差
  • 实数优化：实信号可减少一半计算量
  • 内存访问：连续内存访问比随机访问快
  • 向量化：利用SIMD指令加速
